# v1.1.4

## 修复

- 加固交易所挂牌/交割锁：自动快照调度器在真正创建或轮换快照前，会再次检查实例是否已上架交易所或存在未结交易订单。
- 修复已有自动快照策略在实例挂牌后仍可能被后台定时任务执行的问题，避免挂牌期间生成新快照或轮换旧快照。
- 扩展自动快照调度器守卫和交易所全局守卫，确保后续修改不会绕过交易所实例锁。
